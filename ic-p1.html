สรุปหมวดคำนวณ

ผลตอบแทน · มูลค่าเงินตามเวลา · ความเสี่ยง — จุดที่คนพลาดบ่อยที่สุด เข้าใจสูตร ไม่ใช่จำคำตอบ

① อัตราผลตอบแทน

HPR = [ เงินได้ (ปันผล/ดอกเบี้ย) + (ราคาปลาย − ราคาต้น) ] ÷ ราคาต้น

เช่น ซื้อหุ้น 100 ปันผล 5 ปลายงวดราคา 108 → (5 + 8)/100 = 13%

ค่าเฉลี่ยเลขคณิต (AM) = ΣR ÷ n  |  เรขาคณิต (GM) = [(1+R₁)(1+R₂)…]^(1/n) − 1

GM ใช้กับผลตอบแทนหลายงวด (ทบต้น) และ GM ≤ AM เสมอ ยิ่งผันผวนมาก ยิ่งห่างกัน

E(R) คาดหวัง = Σ [ ความน่าจะเป็น × ผลตอบแทนกรณีนั้น ]

② ผลตอบแทนที่แท้จริง · ภาษี

(1 + nominal) = (1 + real) × (1 + เงินเฟ้อ)  →  โดยประมาณ real ≈ nominal − เงินเฟ้อ
ผลตอบแทนที่ต้องการ = อัตราปราศจากความเสี่ยงแท้จริง + ส่วนชดเชยเงินเฟ้อ + ส่วนชดเชยความเสี่ยง
หลังภาษี = ผลตอบแทน × (1 − อัตราภาษี)  |  TEY = ผลตอบแทนปลอดภาษี ÷ (1 − อัตราภาษี)

TEY ใช้เทียบว่าตราสารปลอดภาษี (เช่นพันธบัตรบางชนิด) เทียบเท่าตราสารที่ต้องเสียภาษีกี่ %

③ มูลค่าเงินตามเวลา (TVM)

FV = PV × (1 + r)ⁿ   |   PV = FV ÷ (1 + r)ⁿ

เช่น ฝาก 1,000 ที่ 6% 3 ปี → 1,000×(1.06)³ = 1,191 บาท

อัตราแท้จริง (Effective) = (1 + i/m)^m − 1   (m = จำนวนงวดต่อปี)

ทบดอกถี่ขึ้น (m มากขึ้น) → effective rate สูงขึ้น เช่น 12%/ปี ทบรายเดือน ≈ 12.68%

④ ความเสี่ยงหลักทรัพย์รายตัว

Variance σ² = Σ [ ความน่าจะเป็น × (R − E(R))² ]  |  SD σ = √Variance

SD (ส่วนเบี่ยงเบนมาตรฐาน) ยิ่งสูง = ความเสี่ยงรวมยิ่งมาก

⑤ ความเสี่ยงกลุ่มหลักทรัพย์ (Portfolio)

ผลตอบแทนพอร์ต = w₁R₁ + w₂R₂ + … (ถ่วงน้ำหนักตามสัดส่วนเงิน)
Covariance = ρ × σ_A × σ_B  |  Correlation ρ = Cov ÷ (σ_A σ_B) , อยู่ระหว่าง −1 ถึง +1
SD พอร์ต 2 สินทรัพย์ = √( w₁²σ₁² + w₂²σ₂² + 2w₁w₂·Cov )

คีย์: ρ ยิ่งต่ำ (เข้าใกล้ −1) การกระจายความเสี่ยงยิ่งได้ผล → SD พอร์ตลดลงมาก

⑥ Beta · CAPM · วัดผลงาน

Beta β = Cov(หุ้น, ตลาด) ÷ Variance ตลาด   (β>1 ผันผวนมากกว่าตลาด)
CAPM : E(R) = R_f + β ( R_m − R_f )

เช่น R_f=2%, β=1.2, R_m=10% → 2 + 1.2(10−2) = 11.6%

Sharpe = (R_p − R_f)/σ_p · Treynor = (R_p − R_f)/β_p · Jensen α = R_p − [R_f + β_p(R_m − R_f)]

Sharpe ใช้ σ (ความเสี่ยงรวม) · Treynor/Jensen ใช้ β (ความเสี่ยงที่เป็นระบบ)

เทคนิคห้องสอบ: จดสูตรพวกนี้ในกระดาษทดทันทีที่เริ่มสอบ (brain dump) ก่อนลืม แล้วค่อยไล่ทำโจทย์
